### Resumen rápido

La forma más fácil de compartir tu código de Colab es:
1. **Copia el código** de tu celda de Colab
2. **Pégalo** en el chat
3. Describe qué necesitas ayuda

Alternativamente, puedes compartir el enlace público de tu notebook de Colab.
